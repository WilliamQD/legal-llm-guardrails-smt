# Privacy and Publication Notes

This public repo deliberately excludes data and course-private artifacts.

## Not Published

- Official held-out legal scenarios.
- Full training datasets.
- Raw model outputs.
- SMT files that encode course-provided cases.
- Adapter checkpoints and generated model artifacts.
- Submission identifiers or grading-specific material.

## Published Instead

- Public-safe code patterns.
- Rewritten method documentation.
- Aggregate result summaries.
- A clear description of the evaluation design.

This keeps the repository useful for recruiters and interviewers while respecting course boundaries and avoiding the release of artifacts that were not intended for public redistribution.
