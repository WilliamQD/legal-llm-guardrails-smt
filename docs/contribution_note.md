# Contribution Note

This was an individual course final project by William Zhang.

My work included:

- translating legal dependency rules into an SMT-checkable structure,
- constructing preference-style examples for LoRA fine-tuning,
- training small language-model adapters under multiple dataset-size settings,
- building the evaluation workflow that parsed model verdicts and compared them against SMT labels,
- analyzing error patterns and writing the final report.

The public repository is a curated version of that work. It removes private data and course materials while preserving the core technical ideas and reusable code patterns.
